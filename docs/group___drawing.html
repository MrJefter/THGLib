Функция очистки экрана

Данная функция используется быстрой очистки экрана в определенный цвет.

Аргументы
fillColorПараметр цвета фона после очистки. Необязателен. Если не указан - фон окрашивается в белый цвет.
brushColorПараметр цвета кисти после очистки. Необязателен. Если не указан - кисть красит в черный цвет.

@usage

#include "TXLib.h"
#include "THGLib.h"
using namespace std;
int main() {
txCreateWindow(600, 354); //Создаем окно 600x354 пикселей
clearScreen(); //Очищаем экран
drawCharacter("Cirno", 0, 0, 1);
Sleep(1000);
clearScreen(RGB (123, 194, 148), RGB (124, 43, 207)); //Очищаем экран в кастомный цвет фона и меняем цвет кисти
drawCharacter("Chen", 0, 0, 2);
}

Функция отрисовки персонажа

Данная функция используется для отрисовки персонажа с указанием его имени, координат на полотне, размера и пеерключаемых параметров по вашему желанию.

Аргументы
characterNameИмя персонажа из списка на главной странице.
xКоордината X левого верхнего угла изображения.
yКоордината Y левого верхнего угла изображения.
sizeМножитель размера персонажа относительно стандартного (300x177).
accessoiresAreVisibleПереключаемый параметр отображения аксессуаров. Необязателен. Если не указан - присваивается значение true.
eyesClosedПереключаемый параметр состояния глаз. Необязателен. Если не указан - присваивается значение false.
mouthOpenedПереключаемый параметр состояния рта. Необязателен. Если не указан - присваивается значение true.
eyebrowVisibleПереключаемый параметр отображения бровей. Необязателен. Если не указан - присваивается значение true.
blushVisibleПереключаемый параметр отображения румянца. Необязателен. Если не указан - присваивается значение true.
hairColorПараметр цвета волос. Необязателен. Если не указан - присваивается стандартное значение для персонажа.
clotherColorПараметр цвета одежды. Необязателен. Если не указан - присваивается стандартное значение для персонажа.
eyesColorПараметр цвета глаз. Необязателен. Если не указан - присваивается стандартное значение для персонажа.

@usage

#include "TXLib.h"
#include "THGLib.h"
using namespace std;
int main() {
txCreateWindow(600, 354); //Создаем окно 600x354 пикселей
clearScreen(); //Очищаем экран
drawCharacter("Cirno", 0, 0, 1); //Рисуем стандартную Сирно на координатах x = 0, y = 0
Sleep(1000);
clearScreen();
drawCharacter("Chen", 0, 0, 2, true, false, false, true, true, RGB (134, 212, 76), RGB (0, 183, 164), RGB (172, 123, 193)); //Рисуем кастомную Чэнь
}
